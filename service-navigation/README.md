# Service Navigation

This module documents two related information systems:

1. A structured community-resource catalog
2. A goal-to-action-to-need pathway model for case-management support

## Core Models

```text
Resource Category
    └── Organization
          └── Service
                ├── Contact
                ├── Location
                ├── Hours
                └── Description
```

```text
Goal
    └── Action Item
          └── Required Need
```

# Dataverse Architecture

This module documents the custom and extended Dataverse data model.

## Verified Scope

- 129 custom tables using the `tr_` publisher prefix
- 94 entities organized within the primary model-driven application
- Custom tables include core business tables, supporting process tables, and relationship/intersect tables
- Standard Microsoft and nonprofit tables were materially configured, extended, or integrated

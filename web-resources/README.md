# Web Resources

This directory will contain selected, reviewed examples of custom JavaScript, HTML, and SVG resources used to extend the model-driven application.

## Publication Requirements

Each example must:

- Be authored or substantially modified by Taylor Askew Ritchie
- Contain no environment-specific secrets or private endpoints
- Include a short explanation of the business problem it solved
- Be reformatted and commented where necessary for public readability
- Preserve the original logic without exposing confidential production details

Planned subdirectories:

- `javascript/`
- `html/`
- `svg/`
